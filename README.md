## How to Install
> ***pip***
- 
  ```shell
  pip install git+https://github.com/kangmg/ase-distortion-interaction.git
  ```

> ***git clone***
- terminal
  ```shell
  ### terminal ###
  git clone https://github.com/kangmg/ase-distortion-interaction.git
  cd ase-distortion-interaction
  pip install .
  ```
